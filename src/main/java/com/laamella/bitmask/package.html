Bitmask collision detection library

This library will give 2D graphics developers, especially game developers, a tool to quickly detect whether two images overlap by comparing their actual visible pixels. This is done by creating an extra datastructure for every image, the Bitmask. It contains the pattern of visible pixels of the image as a pattern of bits. Overlaps can be found very quickly by simply AND-ing two of these bitmasks, supplying an offset.
